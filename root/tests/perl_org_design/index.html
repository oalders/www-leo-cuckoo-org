The Perl Directory

About Perl

Perl is a modern stable, cross platform programming language. It is used for mission critical projects in the public and private sectors.

CPAN

The Comprehensive Perl Archive Network (CPAN) has over 18,000 open source modules available for download.

Perl Core Development

The current version of Perl is 5.10.1, which is under active development. There are several other separate projects including work on Perl 6.

The Perl Foundation

The Perl Foundation is dedicated to the advancement of the Perl programming language through open discussion, collaboration, design, and code.

Learning Perl & Docs

With free online books, over 18,000 extension modules, and a large community, there are many ways to learn and extend your Perl programming skills.

The Perl Community

Perl has a strong active world wide community with over 300 local groups, mailing lists and support/discussion websites.

Books, Blogs and more

Free online books, links to blogs, podcasts and videos.

Events and Conferences

Conferences, social and technical events around the world, at all levels.
