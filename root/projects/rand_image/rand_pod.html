NAME

rand_image.pl - Given a directory (containing images) this script will choose an image at random and create the html to display the image with a link generated from the file name.

SYNOPSIS

        <!--#include virtual="/cgi-bin/rand_image.pl?dir=/images"-->

Where dir is the image directory containing images named so that the A HREF around them uses the name to create an internal (e.g. on the local web site) link.

DESCRIPTION

This script has a specific purpose, it removes the need for a configuration file or altering the code for displaying random image.

The main drawback from this is the lack of an ALT tag in the IMG tag and having to create large file names.

The naming of an image to link to:

/path/to/file.html

would be:

path-to-file.jpg

if the directory parsed to the script was /images

then the file would be

$DOC_ROOT/images/path-to-file.jpg
